> 极客时间算法小课题目练习及相关总结，希望通过这一次的学习巩固对算法的认识更上一层楼，写出更加高效的代码...

## 一周总结（一）
一开始学习算法的动机就是希望可以写出更加高效和简单实用、BUG很少的代码。虽然之前也有过想系统的学习算法的想法，不过都是学习一段时间后就放弃了，因此这么多年过去了依然还是停留在入门阶段，高不成、低不就。

虽然只有一周的时间，而且自己完成的质量和数量都不是很好，做算法题的状态和思考过程也不是很完美。不过，这一次也是重新燃起了心中的希望，打算养成练习算法题、思考算法题的习惯，就像老师说的那样不断的进行基础练习，思考自己的解决方法、看别人的解题思路不断的提高对算法的认识水平。

总的来说这一周的课程完成度不是很好，只看了一期的视频，算法题完成了三个，离五毒还有一定的距离，不过还是得鼓励自己，习惯慢慢养成，这一周就作为一个敲门砖吧。希望在以后的日子里有更大的进步和收获。
